Vous souhaitez contribuer à ce site et nous aider à améliorer le contenu et la
qualité de l'expérience d'apprentissage pour les futur·e·s étudiant·e·s ? Le
code source de ce site est disponible sur GitHub et vous pouvez y contribuer en
soumettant des pull requests ou en signalant des problèmes à l'aide d'issues.

Le lien vers le dépôt GitHub est disponible dans l'en-tête du site. Vous pouvez
également nous contacter directement pour discuter de vos idées et suggestions.

## Partager vos commentaires

Vous souhaitez nous partager un commentaire concernant le cours ? Vos retours
nous permettent d'améliorer le contenu et la qualité de l'expérience
d'apprentissage pour les futurs étudiant··e·s.

Si vous souhaitez, vous pouvez nous laisser vos coordonnées afin que nous
puissions vous contacter pour plus de détails.

Nous nous engageons à ne pas partager vos informations personnelles avec des
tiers et à les utiliser uniquement pour répondre à votre message.

**Pour toutes questions administratives, merci de contacter directement la
personne de contact indiquée dans le e-mail que vous avez reçu pour
l'inscription au cours.**

Merci !
