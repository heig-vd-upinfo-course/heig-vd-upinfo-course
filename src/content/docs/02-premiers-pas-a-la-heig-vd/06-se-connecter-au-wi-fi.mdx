La HEIG-VD met à disposition plusieurs réseaux Wi-Fi sur ses campus. Le réseau
**eduroam** est le principal réseau recommandé pour les étudiant·es.

## eduroam

Eduroam (Education Roaming) est un réseau Wi-Fi sécurisé utilisé par des
milliers d'établissements d'enseignement et de recherche dans le monde entier.
Il vous permet de vous connecter au Wi-Fi non seulement à la HEIG-VD, mais aussi
dans toutes les institutions partenaires (universités, hautes écoles,
bibliothèques, aéroports...) avec les mêmes identifiants.

Pour vous connecter à eduroam à la HEIG-VD :

1. Sélectionnez le réseau **eduroam** dans la liste des réseaux Wi-Fi
   disponibles.
2. Saisissez votre identifiant sous la forme `prenom.nom@hes-so.ch`.
3. Saisissez votre mot de passe HES-SO.
4. Acceptez le certificat si votre appareil vous le demande.

<Aside type="tip">
	La configuration d'eduroam peut varier selon le système d'exploitation. Des
	instructions détaillées par plateforme sont disponibles sur le site du
	helpdesk de la HEIG-VD.
</Aside>

## Autres réseaux disponibles

- **HEIG-VD-Guest** : réseau ouvert à accès limité, destiné aux visiteurs. Il ne
  doit pas être utilisé pour les cours ou les travaux.
- **HEIG-VD** : réseau interne pour les appareils gérés par l'école
  (imprimantes, matériel de laboratoire).

## En cas de problème

Si vous n'arrivez pas à vous connecter à eduroam, vérifiez :

- Que votre compte HES-SO est bien activé (voir le contenu
  [Gérer son compte HES-SO, HEIG-VD et Switch eduID](/heig-vd-upinfo-course/02-premiers-pas-a-la-heig-vd/09-gerer-son-compte-hes-so-heig-vd-et-switch-eduid)).
- Que vous utilisez le bon format d'identifiant (`prenom.nom@hes-so.ch`).
- Que votre mot de passe est correct.

En cas de difficulté persistante, contactez le helpdesk.
