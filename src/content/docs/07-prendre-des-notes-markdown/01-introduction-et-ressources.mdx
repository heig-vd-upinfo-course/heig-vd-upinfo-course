Markdown est un langage de balisage léger qui permet d'écrire du texte mis en
forme en utilisant une syntaxe simple et lisible. Il est très utilisé dans le
développement logiciel : documentation, fichiers `README`, wikis, blogs et prise
de notes.

Apprendre Markdown est un investissement minimal pour un bénéfice durable : vous
l'utiliserez tout au long de votre formation et de votre carrière.

Les thématiques abordées sont les suivantes :

- Présentation de Markdown : la syntaxe de base et les éléments courants.
- Prendre des notes avec Visual Studio Code : l'éditeur recommandé dans ce
  cours.
- Prendre des notes avec Obsidian : une application dédiée à la prise de notes
  en Markdown.
- Prendre des notes avec Pandoc : convertir des documents Markdown en PDF, Word
  ou d'autres formats.
- Alternatives : d'autres outils et approches pour écrire en Markdown.
