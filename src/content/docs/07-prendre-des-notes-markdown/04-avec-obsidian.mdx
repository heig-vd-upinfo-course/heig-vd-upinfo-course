Obsidian est une application dédiée à la prise de notes, basée sur des fichiers
Markdown stockés localement sur votre machine. Elle est gratuite pour un usage
personnel et disponible sur Windows, macOS, Linux, Android et iOS.

## Philosophie : les notes comme graphe

Obsidian se distingue par sa conception des notes en réseau : vous pouvez créer
des **liens entre vos notes** avec la syntaxe `[[Nom de la note]]`. Ces liens
forment un graphe que vous pouvez visualiser, ce qui aide à voir les connexions
entre les idées.

Cette approche est particulièrement utile pour les prises de notes de cours, où
les concepts se référencent mutuellement.

## Télécharger et installer l'application

Utilisez les instructions ci-dessous pour installer l'application sur votre
système d'exploitation. Vous pouvez choisir d'utiliser un gestionnaire de
paquets (recommandé) ou d'installer l'application manuellement.

Téléchargez Obsidian depuis le site officiel :
[obsidian.md](https://obsidian.md)

## Concepts clés

- **Vault** (coffre) : un dossier sur votre disque qui contient toutes vos
  notes. Obsidian travaille directement avec des fichiers `.md` standard.
- **Note** : un fichier Markdown dans le vault.
- **Lien interne** : `[[Titre de la note]]` crée un lien cliquable vers une
  autre note.
- **Tag** : `#mot-clé` permet de catégoriser les notes.
- **Vue graphe** : visualisation de toutes les notes et leurs connexions.

## Fonctionnalités utiles

- **Mode live preview** : éditeur qui affiche le rendu Markdown en même temps
  que la syntaxe.
- **Plugins** : un vaste écosystème de plugins communautaires (calendrier,
  flashcards, diagrammes, etc.).
- **Recherche** : recherche plein texte dans toutes les notes du vault.
- **Templates** : créez des modèles de notes réutilisables.

## Synchronisation

Les notes étant de simples fichiers Markdown, elles peuvent être synchronisées
via :

- **Git** : versionning et synchronisation via GitHub ou GitLab.
- **OneDrive, iCloud, Google Drive** : synchronisation automatique.
- **Obsidian Sync** : service payant officiel avec chiffrement de bout en bout.

## Résumé

Obsidian est une application puissante et flexible pour la prise de notes en
Markdown. Sa capacité à lier les notes entre elles et à les visualiser en graphe
en fait un outil particulièrement adapté aux étudiant·es qui souhaitent
construire une base de connaissances structurée.
