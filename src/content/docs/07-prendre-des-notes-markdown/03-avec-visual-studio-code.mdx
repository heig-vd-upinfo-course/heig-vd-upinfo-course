Visual Studio Code prend en charge Markdown nativement et constitue un excellent
environnement pour rédiger des notes. Il n'est pas nécessaire d'installer quoi
que ce soit pour commencer.

## Prévisualisation Markdown intégrée

VS Code propose une prévisualisation Markdown en temps réel. Pour l'ouvrir :

- Raccourci : `Ctrl+Shift+V` (Windows/Linux) ou `⌘+Shift+V` (macOS).
- Ou cliquez sur l'icône de prévisualisation en haut à droite de l'éditeur
  (icône avec une loupe).

Vous pouvez ouvrir le fichier source et la prévisualisation côte à côte avec
`Ctrl+K V` (ou `⌘+K V`).

## Extensions utiles

Quelques extensions améliorent l'expérience Markdown dans VS Code :

- **Markdown All in One** : raccourcis clavier, table des matières automatique,
  prévisualisation améliorée.
- **Markdown Lint** : vérification du style et de la cohérence de la syntaxe
  Markdown.
- **Prettier** : formatage automatique des fichiers Markdown.
- **Paste Image** : coller une image depuis le presse-papiers directement dans
  un fichier Markdown.

## Organiser ses notes

Avec VS Code, les notes Markdown peuvent être organisées dans un dossier de
votre choix. Une arborescence claire facilite la navigation :

```
notes/
├── cours/
│   ├── algorithmique.md
│   └── reseaux.md
├── projets/
│   └── projet-a.md
└── references.md
```

La recherche globale (`Ctrl+Shift+F`) permet de retrouver rapidement une note
par son contenu.

## Synchronisation avec Git

Un dossier de notes Markdown peut être versionné avec Git et synchronisé sur
GitHub ou GitLab. Cela offre un historique des modifications, une sauvegarde
automatique et la possibilité de travailler sur plusieurs appareils.

## Résumé

VS Code est un environnement polyvalent pour rédiger en Markdown :
prévisualisation intégrée, extensions disponibles et intégration Git native.
C'est un excellent choix si vous souhaitez garder un seul outil pour le code et
les notes.
