En dehors de VS Code, Obsidian et Pandoc, il existe de nombreux autres outils
pour rédiger en Markdown ou prendre des notes. Cette page présente les
alternatives les plus populaires.

## Notion

[Notion](https://www.notion.so) est une application de prise de notes et de
gestion de projets en ligne, très populaire dans les milieux étudiants et
professionnels. Elle supporte une syntaxe proche de Markdown, mais avec une
interface glisser-déposer.

Avantages :

- Interface visuelle intuitive.
- Collaboration en temps réel.
- Bases de données, tableaux Kanban, calendriers intégrés.
- Version gratuite généreuse pour un usage personnel.

Limites :

- Les données sont hébergées sur les serveurs de Notion (aux États-Unis).
- L'exportation des notes peut être moins propre que du Markdown pur.
- Nécessite une connexion Internet pour la plupart des fonctionnalités.

## Zettlr

[Zettlr](https://www.zettlr.com) est un éditeur Markdown open source conçu pour
la recherche académique. Il supporte les références bibliographiques via Zotero
ou JabRef, et est idéal pour rédiger des travaux cités.

## Logseq

[Logseq](https://logseq.com) est une alternative open source à Obsidian, basée
sur des listes imbriques et des liens bidirectionnels. Elle privilégie une
approche jour par jour (journaling) et est très appréciée par les adeptes de la
méthode Zettelkasten.

## Éditeurs en ligne

Si vous avez besoin d'écrire rapidement du Markdown sans rien installer :

- [StackEdit](https://stackedit.io) : éditeur Markdown en ligne avec
  prévisualisation et synchronisation cloud.
- [Dillinger](https://dillinger.io) : éditeur simple et rapide, directement dans
  le navigateur.

## Résumé

Il n'existe pas d'outil universellement meilleur : le choix dépend de vos
besoins, de vos habitudes et de vos exigences en matière de confidentialité.
Pour ce cours, VS Code ou Obsidian sont recommandés. Les autres outils sont à
explorer selon vos préférences personnelles.
