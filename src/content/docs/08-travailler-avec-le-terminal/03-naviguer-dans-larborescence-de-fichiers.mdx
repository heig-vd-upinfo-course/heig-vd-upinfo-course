Naviguer dans l'arborescence de fichiers est la première compétence à acquérir
en terminal. Toutes les autres commandes s'appuient sur elle.

## Commandes de navigation essentielles

### `pwd` - afficher le répertoire courant

```sh
pwd
# /home/alice/Documents
```

`pwd` (Print Working Directory) affiche le chemin absolu du dossier dans lequel
vous vous trouvez actuellement.

### `ls` - lister le contenu d'un dossier

```sh
ls               # Liste les fichiers et dossiers du répertoire courant
ls -l            # Liste détaillée (permissions, taille, date)
ls -a            # Inclut les fichiers cachés (ceux commençant par .)
ls -lh           # Liste détaillée avec tailles lisibles (Ko, Mo...)
ls /etc          # Liste le contenu d'un dossier spécifié
```

### `cd` - changer de répertoire

```sh
cd Documents          # Aller dans le dossier Documents (relatif)
cd /home/alice        # Aller vers un chemin absolu
cd ..                 # Remonter d'un niveau dans l'arborescence
cd ../..              # Remonter de deux niveaux
cd ~                  # Aller dans le dossier personnel
cd -                  # Revenir au répertoire précédent
```

## Chemins absolus et relatifs

Comme vu dans le contenu
[Gestionnaire de fichiers](/heig-vd-upinfo-course/05-configurer-son-systeme-dexploitation-et-ses-applications/03-gestionnaire-de-fichiers),
un **chemin absolu** commence toujours depuis la racine (`/` sur macOS/Linux,
`C:\` sur Windows) et désigne un emplacement fixe.

Un **chemin relatif** est exprimé par rapport au répertoire courant. Les
raccourcis utiles sont :

- `.` : le répertoire courant.
- `..` : le répertoire parent.
- `~` : le dossier personnel de l'utilisateur·trice.

## Autocomplétion avec Tab

La touche `Tab` est votre meilleure allie au terminal. Elle complète
automatiquement les noms de fichiers, dossiers et commandes.

```sh
cd Doc<Tab>  # Complete en cd Documents/ si le dossier existe
```

Si plusieurs correspondances existent, appuyez deux fois sur `Tab` pour les
afficher toutes.

## Historique des commandes

Le terminal conserve un historique des commandes saisies :

- **Flèche haut / bas** : naviguer dans l'historique.
- `Ctrl+R` : recherche dans l'historique (saisissez un mot-clé pour trouver une
  commande précédente).
- `history` : affiche toutes les commandes de l'historique.

## Résumé

Les commandes `pwd`, `ls` et `cd` constituent le trio fondamental de la
navigation en terminal. Maîtriser les chemins absolus et relatifs, ainsi que
l'autocomplétion par `Tab`, rend le travail en ligne de commande fluide et
rapide.
