Quand vous tapez `www.heig-vd.ch` dans votre navigateur, votre ordinateur doit
trouver l'adresse IP correspondante pour établir la connexion. C'est le rôle du
serveur DNS.

## Qu'est-ce que DNS ?

DNS signifie Domain Name System. C'est un système qui traduit les noms de
domaine lisibles par les humains (comme `heig-vd.ch`) en adresses IP comprises
par les machines (comme `193.134.218.50`).

On compare souvent le DNS à un annuaire téléphonique : vous connaissez le nom de
la personne, l'annuaire vous donne son numéro.

## Comment fonctionne une résolution DNS ?

Lorsque vous accédez à un nom de domaine, votre ordinateur effectue une
**résolution DNS** :

1. Il interroge d'abord son **cache local** : si la réponse est déjà connue et
   récente, elle est réutilisée directement.
2. Si le cache ne contient pas la réponse, il interroge un **serveur DNS
   récursif** (souvent fourni par votre fournisseur d'accès ou configuré
   manuellement).
3. Ce serveur interroge à son tour d'autres serveurs DNS dans la hiérarchie
   jusqu'à obtenir l'adresse IP du domaine demandé.
4. La réponse est renvoyée à votre ordinateur, mise en cache, et la connexion
   peut s'établir.

## Serveurs DNS publics

Vous pouvez utiliser des serveurs DNS publics à la place de ceux de votre
fournisseur d'accès. Les plus connus sont :

- `8.8.8.8` et `8.8.4.4` : serveurs DNS de Google.
- `1.1.1.1` et `1.0.0.1` : serveurs DNS de Cloudflare, réputés pour leur
  rapidité et leur respect de la vie privée.
- `9.9.9.9` : serveur DNS de Quad9, axé sur la sécurité.

Changer de serveur DNS peut améliorer la vitesse de résolution ou renforcer
votre confidentialité.

## Résumé

Le DNS traduit les noms de domaine en adresses IP. C'est un service invisible
mais indispensable : sans lui, vous devriez mémoriser des adresses IP pour
accéder à chaque site web.
