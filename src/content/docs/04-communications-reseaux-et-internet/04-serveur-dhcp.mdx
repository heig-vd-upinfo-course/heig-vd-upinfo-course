Lorsque vous vous connectez à un réseau Wi-Fi ou branchez un câble réseau, votre
ordinateur obtient automatiquement une adresse IP sans que vous ayez à la saisir
manuellement. C'est le rôle du serveur DHCP.

## Qu'est-ce que DHCP ?

DHCP signifie Dynamic Host Configuration Protocol. C'est un protocole réseau qui
permet à un serveur d'attribuer automatiquement une configuration réseau aux
appareils qui se connectent. Cette configuration comprend :

- Une **adresse IP** : identifiant unique de l'appareil sur le réseau.
- Un **masque de sous-réseau** : définit l'étendue du réseau local.
- Une **passerelle par défaut** : l'adresse du routeur pour sortir du réseau
  local.
- Des **adresses de serveurs DNS** : pour résoudre les noms de domaine.

## Comment ça fonctionne ?

Le processus d'attribution DHCP se déroule en quatre étapes :

1. **Discover** : l'appareil envoie un message en diffusion sur le réseau pour
   trouver un serveur DHCP.
2. **Offer** : le serveur DHCP répond avec une proposition de configuration
   (adresse IP disponible, durée de bail, etc.).
3. **Request** : l'appareil accepte la proposition et en informe le serveur.
4. **Acknowledge** : le serveur confirme l'attribution. L'appareil peut
   désormais utiliser la configuration.

Cette séquence est souvent résumée par l'acronyme **DORA**.

## Adresse IP statique vs dynamique

Par défaut, les adresses attribuées par DHCP sont **dynamiques** : elles peuvent
changer à chaque reconnexion, selon la disponibilité. C'est suffisant pour la
plupart des usages.

Certains appareils ont besoin d'une adresse **statique** (fixe) qui ne change
pas : un serveur web, une imprimante réseau, un NAS (stockage en réseau). On
peut soit configurer manuellement une adresse IP fixe sur l'appareil, soit
configurer le serveur DHCP pour toujours attribuer la même adresse à un appareil
donné (via son adresse MAC).

## Où se trouve le serveur DHCP ?

Dans un réseau domestique, c'est votre box Internet ou votre routeur qui joue le
rôle de serveur DHCP. Dans un réseau d'entreprise ou universitaire, il s'agit
généralement d'un serveur dédié géré par les équipes informatiques.

## Résumé

Le serveur DHCP automatise l'attribution des adresses IP et de la configuration
réseau sur un réseau. Sans lui, il faudrait configurer manuellement chaque
appareil, ce qui serait peu pratique à grande échelle.
